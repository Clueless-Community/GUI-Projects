# QUIZZO
A Marvel Avengers Game made in Java

## Actual project link [ðŸ”—](https://github.com/MihirRajeshPanchal/Marvel-Avengers-Fighting-Simulator-Game)


## Requirements
- Java 16 and Above

## Setup before run
- To start, run `PlayGame.java` file or 'PlayGameWithMusic.java':
```bash
$ javac PlayGame.java
$ java PlayGame

$ javac PlayGameWithMusic.java
$ java PlayGameWithMusic

```
